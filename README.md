## 🚀 About

CVExplore Maps is a web application that helps users discover nearby places and navigate using Google Maps.

This project was created to improve my skills with APIs, geolocation, JSON data, and Progressive Web Apps (PWA).

---

## 🛠️ Tech Stack

* HTML5
* CSS3
* JavaScript
* APIs
* JSON
* Google Maps
* PWA
* Service Workers

---

## ✨ Features

* Nearby place discovery
* Google Maps integration
* Responsive design
* PWA support
* Fast loading with externally hosted images
